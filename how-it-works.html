Minibloq Software

The programming software enviroment of the robot is Minibloq. Minibloq is a graphical programming enviroment for Aduino and other physical computing devices. Instead of having difficulty out over syntax and the other annoyances of text based codes Minibloq allows you to construct your own program using a series of graphical blocks.
